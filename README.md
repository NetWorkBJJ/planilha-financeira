# Planilha Financeira

Sistema de controle financeiro pessoal com interface web minimalista.

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Selecione o mês e ano desejados
3. Adicione suas entradas e saídas
4. Os dados são salvos automaticamente no localStorage do navegador

## Funcionalidades

- ✅ Registro de entradas e saídas financeiras
- ✅ Cálculo automático de totais e sobra
- ✅ Gráfico visual da distribuição de gastos
- ✅ Seleção de mês/ano
- ✅ Campo de notas para lembretes
- ✅ Persistência local (localStorage)
- ✅ Interface dark mode

## Stack

- HTML5
- CSS3 (design system minimalista)
- JavaScript Vanilla
- Canvas API (gráficos)

## Testando alterações

Após qualquer modificação no código, basta:
1. Salvar o arquivo `index.html`
2. Atualizar a página no navegador (F5 ou Cmd+R)
